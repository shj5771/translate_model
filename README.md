# AI Translation Module for Agent System

이 프로젝트는 **OpenAI GPT 기반 번역 기능을 Agent(에이전트) 시스템에 연결하고 테스트**하기 위해 제작되었음 
복잡한 과정 없이, 간단한 환경에서 번역 기능을 빠르게 구현하고 검증할 수 있도록 구성되었다.

---

## 주요 기능 (Features)

-  OpenAI GPT-4o / GPT-4o-mini 모델 기반 번역
-  한국어 ↔ 영어 등 다국어 번역 지원
-  `test_translation.py` 실행만으로 간단 테스트 가능
-  Agent 시스템에 쉽게 연동 가능한 구조
-  `.env`를 통한 API Key 보안 관리
